PySchool.net is...

  • A python programming environment
  • Solely web based (powered by the Brython transpiler)
  • Students can create, edit, save and execute python code directly in a browser.

PySchool.net directives

  • 100% free for educators and students
  • Reduce the technological burdens of hosting a programming class (all you need is a modern web browser
  • 100% open source (hosted on github, brython-dev/brython-in-the-classroom)
  • Online at PySchool.net

Goals and Challenges

  • Luckily, Brython does most of the heavy lifting..
  • No file system - must emulate a file system, files, basic permissions, etc.
  • Sharing of user defined modules.
  • Submitting of "homework" to grader/educator.
